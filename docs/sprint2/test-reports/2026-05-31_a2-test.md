## 1. 执行范围

### 1.1 被测对象

`core/llm_client.py::resolve_llm_config(llm_config_set, node_name) -> LLMConfig` —— A2 新增的节点级 LLM 路由**纯函数**（不创建 ChatOpenAI、不发网络）。逻辑：
- `llm_config_set` 非 dict / 缺 `default` 键 → 抛 `PermanentError("llm_config_set.default 缺失或形态错误")`
- `node_name` 为 None → 返回 default
- `overrides` 非 dict（含缺失 / None）→ 返回 default
- 否则返回 `overrides.get(node_name, default)`

支持覆写的 4 个合法 NodeName：`paper_intake` / `paper_analysis` / `resource_scout` / `planning`（PRD S2-09 / AC-S2-11）。
